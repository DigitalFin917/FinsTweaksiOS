Infinity Unstick

Fixes Disney Infinity 1.0 on iOS 10 by stopping obsolete SDKs from interfering with launch.

Version 1.0 Disney Infinity: Toy Box Free

Compatibility

  • iOS: 10.0 - 10.3.4
  • Architectures: armv7, arm64
  • Depends: mobilesubstrate
  • Affects: com.disney.DisneyInfinity

Details

Disney Infinity 1.0 on iOS crashes/stalls on iOS 10 because it attempts to call analytics code which no longer functions correctly at runtime. Infinity Unstick NOPs out the implementation for this which restores the same behavior as iOS 6-9.
